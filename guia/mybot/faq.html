Preguntas Frecuentes

Estas son preguntas que se hacen con frecuencia en el servidor soporte en discord MyBOT TEAM, si tiene problemas que no figuran en esta sección, pregunte en el servidor de Discord para poder ayudarlo en su consulta.

¿Porque me aparece un error en la ultima línea de mi código? 

Error: Unexpected token )

Respuesta: este error suele pasar porque no cerraste bien algún comando anterior, fíjate si te falto algún paréntesis ) o alguna llave } o corchete ] por cerrar, este error suele ser muy común cuando estas empezando.


¿Porque me aparece este error en mi consola cuando ejecuto mi bot? 

message is not defined

Respuesta: es por que no has definido o puesto dentro del evento message.


¿Porque me aparece este error en mi consola cuando ejecuto mi bot? 

args is not defined

Respuesta: es por que no has definido la variable 'args' dentro del evento message. Ir al capitulo


¿Como puedo hacer que solo yo pueda usar X comando? 

//Respuesta:
if (message.author.id !== 'IDUSUARIO') return;

Agregamos esta condicional dentro de un comando para validar que solo pueda ser usado por el usuario que fue agregando en 'IDUSUARO'.


¿Como puedo enviar un mensaje con el BOT al privado(MD) de un usuario? 

//Respuesta:
client.users.get('IDUSUARIO').send('MENSAJE')

Usamos la coleccion client.users y usando el metodo get(), ingresamos el id del usuario 'IDUSUARO', para luego usando el metodo send() enviar el mensaje.


¿Como puedo enviar un mensaje aun canal determinado? 

//Respuesta:
client.channels.get('IDCANAL').send('MENSAJE A ENVIAR');

Usamos la coleccion client.channels y usando el metodo get(), ingresamos el id del canal 'IDCANAL', para luego usando el metodo send() enviar el mensaje determinado.


¿Cómo puedo hacer que en mi bot funcione la reproducción de musica?

  • Instale FFMPEG.
  • Para windows: npm install ffmpeg-binaries
  • Para Ubuntu: sudo apt install ffmpeg
